Insert

Untuk Menambah data payout bisa menggunakan method POST pada Api URL Endpoint: https://sodakohku.herokuapp.com/api/
payout/ data yang diperlukan sebagai berikut.

JSON Attribute Type Description
id_user String id dari user yang akan melakukan penarikan dana
nominal Integer Jumlah uang yang akan ditarik. *Minimal 100000
bank_name String Nama bank tujuan. e.g "bca","bni","permata"
nomor_rekening Integer Nomor rekening tujuan
nama_rekening String Nama pemilik rekening

Setelah request selesai. Anda akan mendapat response dari server sebagai berikut:

JSON Attribute Type Description
data Object Array of Json Detail transaksi
status String Status code request possible "success" or "error"
message String Pesan dari server

Example:

  • Python 
import requests 
url = "https://sodakohku.herokuapp.com/api/payout/" 
data = {
    "id_user":"5f2416dd6993ff289ffc5219",
    "nominal":200000,
    "bank_name":"bri",
    "nomor_rekening":999887232323,
    "nama_rekening":"arik setyawan"
}
request = requests.post(url, json=data)

  • Response 
{
  "message": "Payout Berhasil",
  "data": {
    "bank_name": "bri",
    "id_user": "5f2416dd6993ff289ffc5219",
    "nama_rekening": "arik setyawan",
    "nominal": 200000,
    "nomor_rekening": 999887232323,
    "payout_complete_status": false,
    "payout_complete_time": null,
    "payout_number": "PAYOUT46214025290825",
    "payout_request_time": "Tue, 04 Aug 2020 09:30:51 GMT",
    "payout_status": "pending"
  },
  "status": "success"
}

Mark Payout Complete

Untuk membuat data payout terselesaikan bisa menggunakan method GET pada Api URL Endpoint: https://sodakohku.herokuapp.com/api/
mark-payout/ parameter yang diperlukan sebagai berikut.

Params Type Description
payout_id String id dari payout yang akan diproses

Setelah request selesai. Anda akan mendapat response dari server sebagai berikut:

JSON Attribute Type Description
data Object Array of Json Detail transaksi
status String Status code request possible "success" or "error"
message String Pesan dari server

Example:

  • Python 
import requests 
url = "https://sodakohku.herokuapp.com/api/mark-payout/" 
params = {
    "payout_number":"PAYOUT98123207832505"
}
request = requests.get(url, params=data)
