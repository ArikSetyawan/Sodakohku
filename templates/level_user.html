Query

Untuk mengambil data level user bisa menggunakan method GET pada Api URL Endpoint: https://sodakohku.herokuapp.com/api/
level_user/

Example:

  • Python 
import requests 
url = "https://sodakohku.herokuapp.com/api/level_user/" 
request = requests.get(url)

  • Response 
{
    "data": [
        {
            "id": "5f2412d50c9ba5b27d0876c5",
            "nama_level": "admin"
        },
        {
            "id": "5f2416856993ff289ffc5218",
            "nama_level": "Regular"
        }
    ],
    "status": "success"
}

Atau anda bisa menggunakan parameter berikut:

Params Type Description
id_level String Query berdasarkan id

Example:

  • Python 
import requests 
url = "https://sodakohku.herokuapp.com/api/level_user/" 
params = {"id_level":"5f2412d50c9ba5b27d0876c5"}
request = requests.get(url, params=params)

  • Response 
{
  "data": {
    "id": "5f2412d50c9ba5b27d0876c5",
    "nama_level": "admin"
  },
  "status": "success"
}

Insert

Untuk Menambah data level user bisa menggunakan method POST pada Api URL Endpoint: https://sodakohku.herokuapp.com/api/
level_user/ data yang diperlukan sebagai berikut.

JSON Attribute Type Description
nama_level String Nama level user. Harus unik dan tidak boleh kembar.

Setelah request selesai. Anda akan mendapat response dari server sebagai berikut:

JSON Attribute Type Description
status String Status code request possible "success" or "error"
message String Pesan dari server

Example:

  • Python 
import requests 
url = "https://sodakohku.herokuapp.com/api/level_user/" 
data = {"nama_level":"premium"}
request = requests.post(url, json=data)

  • Response 
{
  "message": "Berhasil dibuat",
  "result": "success"
}

Update

Untuk Mengubah data level user bisa menggunakan method PUT pada Api URL Endpoint: https://sodakohku.herokuapp.com/api/
level_user/ data yang diperlukan sebagai berikut.

JSON Attribute Type Description
id_level String id_level user yang akan diubah.
nama_level String Nama level user. Harus unik dan tidak boleh kembar.

Setelah request selesai. Anda akan mendapat response dari server sebagai berikut:

JSON Attribute Type Description
status String Status code request possible "success" or "error"
message String Pesan dari server

Example:

  • Python 
import requests 
url = "https://sodakohku.herokuapp.com/api/level_user/" 
data = {"id_user":"5f2802e61966ae150fbaeb27","nama_level":"Super User"}
request = requests.put(url, json=data)

  • Response 
{
  "message": "level_user Berhasil diupdate",
  "status": "success"
}

Delete

Untuk Menghapus data level user bisa menggunakan method Delete pada Api URL Endpoint: https://sodakohku.herokuapp.com/api/
level_user/ Parameter yang diperlukan sebagai berikut.

Parameters Type Description
id_level String id_level user yang akan dihapus.

Setelah request selesai. Anda akan mendapat response dari server sebagai berikut:

JSON Attribute Type Description
status String Status code request possible "success" or "error"
message String Pesan dari server

Example:

  • Python 
import requests 
url = "https://sodakohku.herokuapp.com/api/level_user/" 
params = {"id_user":"5f2802e61966ae150fbaeb27"}
request = requests.delete(url, params=params)

  • Response 
{
  "message": "level user deleted",
  "status": "success"
}
