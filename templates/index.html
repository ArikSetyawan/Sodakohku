Welcome to Sodakohku Platform API!

Introduction

Sodakohku API adalah sebuah RESTful Web Service untuk penggalangan dana untuk streamer, organisasi, dan perorangan. API ini bisa dimanfaatkan untuk Website atau aplikasi Mobile, sehingga donatur bisa mudah untuk mengakses platform ini kelak. Untuk dokumentasi penggunaan API ini developer sudah menyediakan akses menu di sebelah kanan yang bisa diakses oleh para pengembang Website dan Mobile.

HTTP(S) Request

Sodakohku API bisa diakses menggunakan HTTP(S) request yang menuju ke Sodakohku Base Url endpoint.
Base URL : https://sodakohku.herokuapp.com/api/
