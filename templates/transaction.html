Insert

Untuk Menambah data transaksi bisa menggunakan method POST pada Api URL Endpoint: https://sodakohku.herokuapp.com/api/
transaction/ data yang diperlukan sebagai berikut.

JSON Attribute Type Description
id_user String id dari user yang akan menerima dana
donatur String Nama donatur
email String Email Donatur
nominal Integer Jumlah donasi yang akan diberikan. *Minimal 10000
payment_method String Metode yang akan digunakan untuk donasi. e.g "bca","bni","permata"
pesan String Kata yang akan disampaikan kepada penerima dana

Setelah request selesai. Anda akan mendapat response dari server sebagai berikut:

JSON Attribute Type Description
data Object Array of Json Detail transaksi
status String Status code request possible "success" or "error"
message String Pesan dari server

Example:

  • Python 
import requests 
url = "https://sodakohku.herokuapp.com/api/transaction/" 
data = {
    "id_user":"5f2416dd6993ff289ffc5219",
    "donatur":"Hamba Allah",
    "nominal":200000,
    "pesan":"Semangat Kakak!!",
    "email":"koechenggarroeng@gmail.com",
    "payment_method":"permata"
}
request = requests.post(url, json=data)
