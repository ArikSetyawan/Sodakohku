Query

Untuk mengambil data user bisa menggunakan method GET pada Api URL Endpoint: https://sodakohku.herokuapp.com/api/
user/

Example:

  • Python 
import requests 
url = "https://sodakohku.herokuapp.com/api/user/" 
request = requests.get(url)

  • Response 
{
  "data": [
    {
      "email": "mastya@gmail.com",
      "email_verify": false,
      "id_level": "5f2412d50c9ba5b27d0876c5",
      "id_user": "5f2416dd6993ff289ffc5219",
      "nama_level": "admin",
      "password": "inimastya",
      "saldo": 0,
      "username": "MasTya"
    },
    {
      "email": "testingSodakoh@gmail.com",
      "email_verify": false,
      "id_level": "5f2416856993ff289ffc5218",
      "id_user": "5f2808ee49a26f11e2bb0192",
      "nama_level": "Regular",
      "password": "12345678",
      "saldo": 0,
      "username": "UserSodakoh123"

  ],
  "status": "success"
}

Atau anda bisa menggunakan salah satu parameter berikut:

Params Type Description
id_user String Query berdasarkan id
username String Query berdasarkan username

Example Query by id:

  • Python 
import requests 
url = "https://sodakohku.herokuapp.com/api/user/" 
params = {"id_user":"5f2808ee49a26f11e2bb0192"}
request = requests.get(url, params=params)

  • Response

Insert

Untuk Menambah data user bisa menggunakan method POST pada Api URL Endpoint: https://sodakohku.herokuapp.com/api/
user/ data yang diperlukan sebagai berikut.

JSON Attribute Type Description
id_level String Hak akses yang akan diberikan.
email String Email user yang akan ditambahkan
username String Username unik
password String Password user yang akan ditambahkan

Setelah request selesai. Anda akan mendapat response dari server sebagai berikut:

JSON Attribute Type Description
status String Status code request possible "success" or "error"
message String Pesan dari server

Example:

  • Python 
import requests 
url = "https://sodakohku.herokuapp.com/api/user/" 
data = {
    "id_level":"5f2416856993ff289ffc5218",
    "email":"testingSodakoh@gmail.com",
    "username":"UserSodakoh123",
    "password":"12345678"
}
request = requests.post(url, json=data)

  • Response 
{
  "message": "user created",
  "status": "success"
}

Update

Untuk Mengubah data user bisa menggunakan method PUT pada Api URL Endpoint: https://sodakohku.herokuapp.com/api/
user/ data yang diperlukan sebagai berikut.

JSON Attribute Type Description
id_user String id dari user yang akan diubah.
username String Username baru Harus unik dan tidak boleh kembar.
password String Password baru Harus unik dan tidak boleh kembar.

Setelah request selesai. Anda akan mendapat response dari server sebagai berikut:

JSON Attribute Type Description
status String Status code request possible "success" or "error"
message String Pesan dari server

Example:

  • Python 
import requests 
url = "https://sodakohku.herokuapp.com/api/user/" 
data = {
    "id_user":"5f2808ee49a26f11e2bb0192",
    "username":"UserBaruSodakoh123",
    "password":"99995555"
}
request = requests.put(url, json=data)

  • Response 
{
  "message": "User Updated",
  "status": "success"
}

Delete

user/ Parameter yang diperlukan sebagai berikut.

Parameters Type Description
id_user String id dari user yang akan dihapus.

Setelah request selesai. Anda akan mendapat response dari server sebagai berikut:

JSON Attribute Type Description
status String Status code request possible "success" or "error"
message String Pesan dari server

Example:

  • Python 
import requests 
url = "https://sodakohku.herokuapp.com/api/user/" 
params = {"id_user":"5f2808ee49a26f11e2bb0192"}

  • Response 
{

  "status": "success"
}
